# FocusLab Habit Tracker

A Next.js habit tracking webapp with calendar-excel design for daily habit management and progress visualization.

## Features

- Calendar-grid layout for habit tracking
- Add/manage multiple habits
- Daily checkbox tracking
- Progress chart visualization
- Responsive design

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Recharts
- CSS Grid
